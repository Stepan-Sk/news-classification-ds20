# Сбор и анализ новостных данных

## Описание проекта
Проект посвящен сбору корпуса новостных текстов из открытых источников СМИ и последующему анализу и построению модели классификации новостей по тематическим классам.

## Цель работы
Собрать размеченный датасет новостей и исследовать методы автоматической классификации новостных публикаций по рубрикам.

## Постановка задачи
Необходимо:
1. Собрать новости из открытых источников СМИ.
2. Сформировать размеченный датасет, включающий текст новости, дату публикации, источник и тематическую рубрику.
3. Использовать рубрики СМИ в качестве целевых классов для задачи классификации.
4. Провести предварительный анализ данных предобработку данных.
5. Построить модели для автоматической классификации новостей по рубрикам.
6. Сравнить качество подходов по выбранным метрикам и сделать выводы.

## Предполагаемая структура данных
- source — источник
- url — ссылка на новость
- published_at — дата и время публикации
- title — заголовок
- text — текст новости
- category_raw — исходная рубрика на сайте СМИ
- category_norm — нормализованный класс

## Предполагаемые классы
- politics
- economics
- society
- sports
- technology
- culture
- world

## План выполнения
1. Выбор источников новостных данных и описание схемы классов.
2. Разработка парсеров и сбор корпуса новостей.
3. Очистка данных, дедупликация и нормализация рубрик.
4. Подготовка итогового датасета для обучения и тестирования моделей.
5. Исследовательский анализ данных.
6. Построение моделей классификации.
7. Оценка качества моделей и сравнение результатов.
8. Подготовка текста работы и презентации.

## Ближайшие шаги
- создать структуру проекта;
- зафиксировать список источников;
- собрать первые данные;
- сформировать первую версию датасета для обучения.
